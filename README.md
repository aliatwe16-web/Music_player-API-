# Pulse Player 🎵

A modern Spotify-style music player built with **HTML, CSS, and Vanilla JavaScript**.

## Features

✅ Modern responsive UI

✅ Search songs by title or artist

✅ Play / Pause controls

✅ Next / Previous track

✅ Volume control

✅ Progress bar with seeking

✅ Song duration display

✅ Album artwork display

✅ Playlist sidebar

✅ Album cover rotation animation while playing

✅ Smooth hover and click animations

✅ Mobile-friendly design

✅ API error handling

✅ No frameworks required
